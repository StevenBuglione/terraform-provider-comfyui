# comfyui_workspace_validation (Data Source)

Validates native ComfyUI workspace JSON by translating it to prompt JSON and checking it against live /object_info metadata.

Use executable validation for workspace JSON that should translate into a runnable workflow now, and `mode = "fragment"` when validating incomplete editor artifacts or subgraph fragments during authoring.

## Example Usage

```terraform
data "comfyui_workspace_validation" "example" {
  mode = "fragment"
  json = jsonencode({
    version = 1
    id      = "workspace-example"
    name    = "Workspace Example"
    nodes   = []
    links   = []
    groups  = []
    config  = {}
    extra   = {}
    definitions = {
      subgraphs = []
    }
  })
}

output "workspace_valid" {
  value = data.comfyui_workspace_validation.example.valid
}
```
