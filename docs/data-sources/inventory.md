# comfyui_inventory (Data Source)

Returns live ComfyUI runtime inventory values for built-in dynamic model categories such as checkpoints and LoRAs.

Use `comfyui_inventory` when generated node schema reports a dynamic inventory-backed input such as checkpoints or LoRAs. It lets you inspect the live values on the target ComfyUI server before authoring or applying a workflow.

## Example Usage

```terraform
data "comfyui_inventory" "example" {
  kinds = ["checkpoints", "loras"]
}

output "inventories" {
  value = data.comfyui_inventory.example.inventories
}
```
