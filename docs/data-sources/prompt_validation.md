# comfyui_prompt_validation (Data Source)

Validates native ComfyUI prompt JSON against live /object_info metadata.

Use executable validation for prompt JSON that is supposed to run now, and `mode = "fragment"` for intentionally incomplete prompt fragments that are still being edited or synthesized.

## Example Usage

```terraform
data "comfyui_prompt_validation" "example" {
  mode = "fragment"
  json = jsonencode({
    "1" = {
      class_type = "LoadImage"
      inputs = {
        image = "example.png"
      }
    }
  })
}

output "prompt_valid" {
  value = data.comfyui_prompt_validation.example.valid
}
```
