# comfyui_workspace_to_terraform (Data Source)

Translates ComfyUI workspace JSON to prompt JSON, then synthesizes canonical Terraform IR and rendered HCL.

Use this data source when you start from editor-native workspace JSON and want the provider to own both the workspace-to-prompt translation and the final Terraform synthesis. The returned fidelity and notes explain any gaps or synthesized structure.

## Example Usage

```terraform
data "comfyui_workspace_to_terraform" "example" {
  name = "workspace-synthesis"
  workspace_json = jsonencode({
    version = 1
    id      = "workspace-example"
    name    = "Workspace Example"
    nodes   = []
    links   = []
    groups  = []
    config  = {}
    extra   = {}
    definitions = {
      subgraphs = []
    }
  })
}

output "terraform_ir_json" {
  value = data.comfyui_workspace_to_terraform.example.terraform_ir_json
}
```
